# superstore-sales-dashboard

Autor: Roddy R. G.

Projeto criado como parte de um desafio de análise de dados e construção de dashboard em Excel.

27NOV2025

------------------------------------------------------------
DESAFIO:

Criar um dashboard de vendas no Excel, com foco na organização e visualização de dados.

O objetivo é transformar dados brutos em informações visuais claras e úteis, permitindo uma análise eficaz
do desempenho de vendas e a tomada de decisões baseadas em dados.

------------------------------------------------------------
DADOS:

Fonte pública gratuita Kaggle Datasets.

Superstore Dataset

Dataset containing Sales & Profits of a Superstore.

https://www.kaggle.com/datasets/vivek468/superstore-dataset-final/data

Conjunto de dados contendo informações relacionadas a vendas, lucros e outros dados relevantes de uma grande rede de supermercados.

Arquivo CSV com 21 colunas e 9994 linhas.

Possui informações do Pedido, do Cliente, Geográficas e Financeiras.

Coluna -------> Descrição do Tipo	
- Row ID: 		ID único para cada linha.
- Order ID: 	ID único do pedido para cada cliente.
- Order Date: 	Data do pedido do produto.
- Ship Date:	Data de envio do produto.
- Ship Mode: 	Modo de envio especificado pelo cliente.
- Customer ID:	ID único para identificar cada cliente.
- Customer Name:	Nome do cliente.
- Segment:		Segmento ao qual o cliente pertence.
- Country:		País de residência do cliente.
- City:		Cidade de residência do cliente.
- State:		Estado de residência do cliente.
- Postal:		Código postal de cada cliente.
- Region:		Região à qual o cliente pertence.
- Product ID: 	ID único do produto.
- Category: 	Categoria do produto encomendado.
- Sub-Category: 	Subcategoria do produto encomendado.
- Product Name: 	Nome do produto.
- Sales: 		Valor das vendas do produto.
- Quantity: 	Quantidade do produto.
- Discount: 	Desconto concedido.
- Profit: 		Lucro/Prejuízo obtido.
